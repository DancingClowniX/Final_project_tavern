Тема проекта: Средневековая таверна (Ресторан+магазин мерча+концертный зал)

Функционал:
1.	Клиентская часть:
●	Главная страница с баннерами, предложениями и новинками.
●	Меню с подаваемыми блюдами с ценой, категорией блюда, с возможностью фильтрации по категориям.
●	Страницы отдельных блюд с подробным описанием, фотографиями и отзывами посетителей.
●	Страница корзины для просмотра выбранных товаров из магазина мерча и оформления заказа.
●	Личный кабинет пользователя с историей заказов, настройками профиля и возможностью отслеживания статуса заказа.
●	Страница с мероприятиями (рыцарский турнир, выступление бардов) с возможностью бронирования места.

2.	Серверная часть:
●	Регистрация и аутентификация пользователей.
●	Взаимодействие с СУБД для хранения информации о блюд, пользователях, заказах и отзывах.
●	Обработка заказов: добавление в базу данных выбранного мерча (шарф, платок итд, итп), расчет итоговой стоимости, отправка уведомлений пользователю.
●	Административная панель для управления блюдами, заказами и пользователями.
●	Обработка и валидация форм на стороне сервера для предотвращения ошибок и атак.
●	Функционал бронирования мероприятия.
